## 数值运算
- += -= 对运算后的值有隐式类型转化（转化为左侧引用的类型）
- %：取模
- +
- -
- *
- /

## 位运算
- &：按位与、逻辑与
- |：安位或
  
## 逻辑运算
- &：逻辑与：两侧表达式均执行
- &&：短路与运算，左侧如果为 flase 则不执行右侧代码。
- |：逻辑或：两侧表达式均执行
- ||：短路或运算，如果左侧为 true 则不执行右侧代码。

## equals&hashCode
- 关系
  - 两个对象相等，hashCode 一定相同（约定）
  - hashCode 相同，两个对象不一定相等
- equals
  - 比较两个对象是否相等
  - 默认使用 == 进行比较
- hashCode：
  - 计算对象的哈希值
